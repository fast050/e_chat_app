# E Chat App
Chatting App

## - System Design

```plaintext
lib/
│
├── core/
│   ├── DI (Dependency Injection)
│   ├── Networking (example: Dio, Retrofit, SDK, ...)
│   ├── Routing
│   ├── Helpers
│   ├── Theming
│   └── Widgets
│
└── features/
    └── home/
        ├── data/
        │   ├── models/
        │   └── repo/
        │
        ├── logic (example: cubit/state, ...)
        │
        └── ui/
            ├── screens/
            └── widgets/
```
